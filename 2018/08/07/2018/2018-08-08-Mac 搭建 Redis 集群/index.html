总结

手动搭建 Redis Cluster 还是很麻烦的，需要下载，配置文件，启动节点，管理节点，分配 slot，管理主从复制等。

Redis 提供了一个小工具：redis-trib.rb，擦用 ruby 编写，内部也是通过这些命令的，直接使用命令有助有我们理解 Redis Cluster 的原理。
