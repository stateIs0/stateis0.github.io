总结

首先第一感觉是，这个框架还是挺好用，挺简单的，基于当前的 SpringBoot 。快速启动。而且不是 SpringCloud 的 Http 调用，使用 Netty 作为网络通信框架，性能当然是没有问题的。

当然，我们这里的 demo 使用的注册中心没有使用 ZK，毕竟初体验嘛，使用的本地的文件。

然而，楼主对这个框架有很大的兴趣。接下来的空闲时间里，楼主将好好研究 SOFA 相关的代码。
